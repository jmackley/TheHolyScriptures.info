Current features of TheHolyScriptures.info

  • Direct lookup: Navigate to specific scriptures using the accordian style interface
  • Searching: Search one or more volumes using advanced search criteria.
  • Updates: Direct inline updating for commentary (requires password)
  • Compare similar verses: Side by side comparison of scripture showing delta differences with markup. (Example: 3 Nephi 13 vs Matthew 6)
  • Scripture font sizing: Manually increase and decrease sizes.
  • Additional information for selected verses: Very limited (Examples: Matthew 1, Genesis 1)
Future features of TheHolyScriptures.info
  • Multiple versions: Unicode Greek and Hebrew originals as well as JST versions of the Bible.
  • Language Support: Support French, Spanish, German.
  • Scripture markup and tagging: Support user highlighting and notes attached to various texts.
  • Enhanced searching: Better interface and additional options for searching. Persisting search criteria for each visitor to reuse later.
  • Statistical search results: Present search statistics.
  • Study aid helps: Add an API into scriptures.lds.org, if it is ever supported (I know it's been discussed on tech.lds.org).
  • Printing support: Present options for printing
  • Additional information for verses: More commentary and multi-media.
