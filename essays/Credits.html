Credits/Acknowledgements

The site runs with Apache (v1.3.41) on Linux and is built with MySQL (v5.0.67), JavaScript, XML, CSS (including CSS3), xhtml (strict), PHP (v5.2.0) and AJAX techniques.

MySQL: The original scripture database comes courtesy of Steve Dibb and his Documentation Project. For my purposes, the original database has been expanded with additional tables and columns.

JavaScript: There is heavy use of JavaScript. The foundational libraries include Prototype Javascript Framework, Scriptaculous, Accordion 2.0 and LyteBox. The foundational libraries comprise 85% of the JavaScript with 15% as custom code.

PHP: PHP is used mostly on the back-end to service AJAX calls by querying the database and returning XML documents. No PHP libraries are used as it is all custom coded.

Development tools: Programmers Notepad 2, Firefox browser with extensions: Firebug (with YSlow plugin), Venkman JavaScript debugger, JavaScriptLint syntax checker, HTML Validator, and ColorZilla.
